# Learning Command

Review project-specific Otto assets in `otto/rules/` and `otto/commands/` against the standards in `otto/rules/learning.md`.

## Steps

1. Read `otto/rules/learning.md`
2. Inspect project-specific rules and commands
3. Identify any rule or command that is unclear, duplicative, conflicting, or stale
4. Summarize recommended improvements, prioritizing rules tied to validated project patterns
